- 숫자 하나를 문자열로 입력할 경우 해당 숫자를 반환한다.
- 기본 구분자인 , : 를 가지는 문자열을 전달하는 경우 구분자를 기준으로 분리한 각 숫자의 합을 반환한다.
  - 컴마(,) 구분자로 입력할 경우 숫자의 합을 반환한다. 예) "1,2" => 3
  - 콜론(:) 구분자로 입력할 경우 숫자의 합을 반환한다.  예) "1:2" => 3, “1,2:3” => 6
- “//”와 “\n” 사이에 위치하는 문자를 커스텀 구분자로 사용할 수 있다. 예)  “//;\n1;2;3” -=> 6
- 빈 문자열 또는 null값을 입력할 경우 0을 반환한다.
- 숫자 이외의 값 또는 음수를 전달하는 경우 RuntimeException 예외가 발생한다. 예 : “-1,2,3”
