## 리팩토링 요구사항
핵심 비지니스 로직을 가지는 객체를 domain 패키지, UI 관련한 객체를 view 패키지에 구현한다.
MVC 패턴 기반으로 리팩토링해 view 패키지의 객체가 domain 패키지 객체에 의존할 수 있지만, domain 패키지의 객체는 view 패키지 객체에 의존하지 않도록 구현한다.

## 기능 요구사항
- [x] UI 관련한 객체를 view 패키지로 변경한다.
- [ ] MVC 패턴 기반으로 리팩토링해 view 패키지의 객체가 domain 패키지 객체에 의존할 수 있도록 리팩토링한다.
- [ ] domain 패키지의 객체는 view 패키지 객체에 의존하지 않도록 리팩토링한다.
- [ ] Random 값 생성 처리에 대한 리팩토링을 진행한다.
