## 기능 요구사항
- 자동차에 이름을 부여할 수 있다.
  - 자동차의 이름은 5자를 초과할 수 없다.
- 전진하는 자동차를 출력할 때 자동차의 이름을 같이 출력한다.
- 자동차의 이름은 쉼표를 기준으로 구분한다.
  - 쉼표가 아닌 경우 예외처리를 한다
- 자동차 경주 게임을 완료한 후 누가 우승했는지 알려준다. 우승자는 한명 이상일 수 있다.

## 프로그래밍 요구사항
- indent의 depth는 1까지만 허용한다.
- 함수의 길이가 15라인을 넘지 않도록 한다.
- 모든 로직에 단위 테스트를 구현한다.
- 자바 코드 컨벤션을 지킨다.
  - 코드 포맷팅도 신경쓰자
- else 예약어를 사용하지 않는다.
