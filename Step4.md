# 자동차 경주 요구사항
## 4단계 기능 요구사항

### 입력
- [x] 사용자가 자동차의 이름 입력한다.
  - [x] 자동차의 이름은 쉼표(,)를 기준으로 구분된다.
- [x] 사용자가 시도 횟수를 입력한다

### 자동차
- [x] 자동차는 이름과 위치 정보를 가진다.
  - [x] 자도앛의 이름은 5자를 초과할 수 없다.
- [x] 전달받은 랜덤 값이 4 이상이면 자동차가 전진한다
- [x] 전달받은 랜덤 값이 4 미만이면 자동차가 정지한다
- [x] 자동차는 전진할 때 1씩 이동한다

### 레이싱게임
- [x] 입력받은 이름의 개수만큼 자동차를 생성한다.
- [x] 입력받은 횟수만큼 라운드를 반복한다
- [x] 게임에서 매 라운드마다 자동차로 0에서 9 사이의 랜덤 값을 전달한다
- [x] 라운드마다 모든 자동차가 이동을 시도한다
- [ ] 라운드가 끝날 때마다 모든 자동차의 현재 위치를 출력한다

### 출력
- [x] 각 자동차의 이름과 위치를 `-`로 표시한다 ex) yang : ---
- [x] 이동 거리n 만큼 `-`를 출력한다
- [x] 라운드의 결과를 개행으로 구분하여 출력한다

### 우승자
- [ ] 모든 라운드가 끝나면 가장 멀리 이동한 자동차가 우승이다.
- [ ] 우승자는 여러명일 수 있으며 하나 이상인 경우 모두 표시한다.
