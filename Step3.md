# 자동차 경주  요구사항
## 기능 요구사항

### 입력
- [ ] 사용자가 자동차 수를 입력한다
- [ ] 사용자가 시도 횟수를 입력한다

### 자동차
- [ ] 각 자동차는 0에서 9 사이의 랜덤 값을 생성한다
- [ ] 생성한 값이 4 이상이면 자동차가 전진한다
- [ ] 생성한 값이 4 미만이면 자동차가 정지한다
- [ ] 자동차는 전진할 때 1씩 이동한다

### 게임
- [ ] 입력받은 횟수만큼 라운드를 반복한다
- [ ] 라운드마다 모든 자동차가 이동을 시도한다
- [ ] 라운드가 끝날 때마다 모든 자동차의 현재 위치를 출력한다

### 출력
- [ ] 각 자동차의 위치를 `-`로 표시한다
- [ ] 이동 거리n 만큼 `-`를 출력한다
- [ ] 라운드의 결과를 개행으로 구분하여 출력한다
