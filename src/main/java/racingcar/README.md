## step 3 - 자동차 경주

### 요구사항

#### 비지니스 로직

##### 자동차 

- [ ] 이동하다 - 0~9 사이의 랜덤값을 구한 후 4이이면 이동한다. 이동했다면 이동 횟수를 갱신한다.
- [ ] 자동차의 현재 이동 횟수를 얻는다 - 이동 횟수를 리턴한다.

##### 레이싱게임

- [ ] 진행하다 - 입력된 시도횟수만큼 게임을 진행한다.

#### UI 입출력

##### InputView

- [ ] 자동차 대수와 시도할 횟수를 입력 받는다.

##### ResultView

- [ ] 자동차의 이동 횟수를 매 시도마다 출력한다.
