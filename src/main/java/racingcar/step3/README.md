 -
   **- 요구사항**
   1. n대의 자동차는 전진 or 정지 가능

   3. 전진 조건 : 0~9 랜덤값에서 4이상일 경우 
   4. 자동차 상태를 화면에 출력, 어느 시점에 출력 제한 없음
