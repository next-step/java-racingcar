
## 기능 목록
- **자동차 클래스 (Car.java)**
    - 자동차의 현재 위치를 관리한다.
    - 이동 조건에 따라 위치를 갱신한다.

- **이동 전략 인터페이스 (MovingStrategy.java)**
    - 자동차가 이동할 수 있는지 여부를 결정하는 전략을 정의한다.

- **랜덤 이동 전략 (RandomMovingStrategy.java)**
    - 0부터 9까지의 랜덤 숫자를 생성하고, 숫자가 4 이상일 경우 이동한다.

- **자동차 경주 (Race.java)**
    - 사용자가 지정한 자동차 대수 만큼 자동차 객체를 생성한다.
    - 각 라운드마다 각 자동차에 대해 이동 여부를 결정하여 진행한다.

- **UI 로직**
    - **InputView.java**: 사용자로부터 자동차 대수와 경주 횟수를 입력받는다.
    - **ResultView.java**: 각 라운드 이후 자동차의 이동 상태를 대시(`-`)로 출력한다.

- **단위 테스트**
    - **CarTest.java**: 자동차 이동 기능에 대한 테스트를 구현한다.
    - **RaceTest.java**: 경주 진행 로직에 대해 예측 가능한 랜덤 값을 사용하여 테스트를 수행한다.
