# 🚗 RacingCarGame
---
## 기능 요구사항

* 초간단 자동차 경주 게임을 구현합니다.

* 자동차 이동 조건: 주어진 횟수 동안 n대의 자동차는 전진하거나 멈출 수 있습니다.

* 사용자 입력: 사용자는 몇 대의 자동차로 몇 번의 이동을 할 것인지 입력할 수 있어야 합니다.

* 전진 조건: 0~9 사이의 난수를 생성한 후, 값이 4 이상이면 자동차가 이동합니다.

* 결과 출력: 자동차의 상태를 화면에 출력합니다. 출력 시점에 대한 제약은 없습니다.

## 프로그래밍 요구사항

* 단위 테스트 구현: 모든 로직에 단위 테스트를 작성해야 합니다. 단, UI(System.out, System.in) 로직은 제외합니다.

* 핵심 로직을 구현하는 코드와 UI 로직을 분리합니다.

* UI 처리를 담당하는 InputView, ResultView 클래스를 별도로 구현합니다.

* 자바 코드 컨벤션 준수: 일관된 스타일을 유지합니다.

* else 예약어 사용 금지: 불필요한 else 문을 제거하고, 명확한 조건 분기를 사용합니다.
