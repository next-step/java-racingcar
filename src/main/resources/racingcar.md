### 🚀 3단계 - 자동차 경주

#### 기능 요구사항

- [x] 사용자 입력값으로 자동차 수와 이동 횟수를 입력받는다.
- [x] 입력된 자동차 수 만큼 자동차를 생성한다.
- [x] 자동차별 0에서 9 사이의 랜덤값을 얻는다.
- [x] 랜덤값이 4 이상일 경우 한 칸 전진한다.
- [x] 랜덤값이 4 미만일 경우 정지한다.
- [x] 입력된 이동 횟수만큼 반복한다.
- [x] 이동 횟수만큼 반복하면서 자동차 위치를 표시한다.
