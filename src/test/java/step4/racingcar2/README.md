# 🚀 3단계 - 자동차 경주(우승자) [TEST]

## # TEST 목록

* `RacingCarName`
  * 5글자 이하 이름
  * 5글자 초과 이름

* `RacingCarEngine`
  * 고장난 엔진(앞으로 못가는) 
  * 베스트 엔진(항상 앞으로 가는)

* `RacingCar`
  * 고장난 엔진을 달았을 때 앞으로 못가는지
  * 베스트 엔진을 달았을 때 항상 앞으로 가는지

* `RacingCarGameRound`
  * 우승자를 찾기

* `InputView`
  * 자동차 이름들 입력받기
  * 시도 횟수 입력받기

* `ResultView`
  * 라운드 결과 출력하기
  * 우승자 출력하기
