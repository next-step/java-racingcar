# 자동차 경주 게임
## 기능 목록

1. **사용자 입력 처리**:
    - 사용자가 자동차의 대수와 이동 횟수를 입력할 수 있도록 합니다.
    - 입력된 값은 유효성 검사를 통해 올바른 범위인지 확인합니다.

2. **자동차 이동 로직**:
    - 각 자동차는 주어진 횟수 동안 전진하거나 멈출 수 있습니다.
    - 전진 조건: 0에서 9 사이의 랜덤 값이 4 이상일 경우 전진합니다.

3. **자동차 상태 출력**:
    - 게임 진행 중에 자동차의 현재 상태를 화면에 출력합니다.
    - 출력 시점은 개발자가 자유롭게 결정할 수 있습니다.

4. **게임 종료 조건**:
    - 사용자가 입력한 이동 횟수가 끝나면 게임이 종료됩니다.
    - 게임 종료 시 최종 결과를 출력합니다.

5. **오류 처리**:
    - 잘못된 입력이나 예상치 못한 오류가 발생할 경우 적절한 오류 메시지를 출력합니다.
