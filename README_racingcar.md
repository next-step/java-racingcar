# 요구사항 정리
## InputView
* 사용자에게 자동차 갯수를 입력 받음
* 사용자에게 주어진 횟수를 입력 받음

## Car
* 자동차의 위치를 전달
* 자동차의 위치를 업데이트
  * 0~9 랜덤 숫자를 뽑아서 4 이상이면 전진

## CarGroup
* 자동차 배열에 입력받은 갯수만큼 자동차 생성
* 자동차 배열의 길이 전달
* 자동차 배열 내의 모든 자동차 위치를 업데이트
* 자동차 배열 내의 자동차 위치 전달

## Racing
* 경주 진행
  * 주어진 횟수만큼 경주를 반복
  * 차의 위치를 출력

## OutputView
* 자동차 하나의 위치를 출력
* 모든 자동차의 위치를 출력
