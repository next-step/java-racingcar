# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 코드 리뷰 요청을 한다.
* 코드 리뷰 피드백에 대한 개선 작업을 하고 다시 PUSH한다.
* 모든 피드백을 완료하면 다음 단계를 도전하고 앞의 과정을 반복한다.

## 기능 목록

정책
- 자동차는 0부터 9까지의 수 중에서 랜덤한 수를 받는다.
- 랜덤 값이 4 이상일 경우 자동차는 전진한다.
- 랜덤 값이 4 미만일 경우 자동차는 정지한다.
- 자동차는 전진할 때마다 위치를 1씩 증가시킨다.
- 자동차의 위치는 '-' 문자로 표현한다.
- 주어진 횟수만큼 모든 자동차는 전진 또는 멈춤을 반복한다.


단위 테스트
- [x] 자동차의 초기 위치는 0이다.
- [x] 자동차는 허용된 범위 중에서 4 이상인 수를 받으면 전진한다.
- [x] 자동차는 허용된 범위 중에서 4 미만인 수를 받으면 멈춘다.
- [x] 자동차가 전진하지 않았을 때 빈 문자열을 반환한다.
- [x] 자동차가 전진했을 때 위치에 해당하는 개수만큼 "-"를 반환한다.
- [x] 자동차가 여러 번 전진하면 전진한 횟수만큼 위치가 증가한다.
- [x] 입력 값이 허용된 수가 아닐 경우 IllegalArgumentException 예외를 발생시킨다.
