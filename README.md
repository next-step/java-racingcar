# 🚗 자동차 경주 게임

## 진행 방법

* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 코드 리뷰 요청을 한다.
* 코드 리뷰 피드백에 대한 개선 작업을 하고 다시 PUSH한다.
* 모든 피드백을 완료하면 다음 단계를 도전하고 앞의 과정을 반복한다.

## 구현 기능 목록

### 🚀 3단계 - 자동차 경주

- [X] 사용자로부터 입력을 받기 위해 입력 정보에 대한 메시지를 출력한다.
- [ ] 사용자로부터 몇 대의 자동차로 몇 번의 이동을 할 것인지를 입력받는다.
    - [ ] 입력값이 정수가 아니거나, 1보다 작은 정수일 경우 RuntimeException 을 반환한다.
- [ ] 0에서 9 사이에서 random 값을 구한 후 random 값이 4이상일 경우일 때만 자동차가 전진할 수 있다.
- [ ] 매 라운드 마다 모든 자동차를 이동한다.
- [X] 각 라운드 실행에 따른 각 자동차의 이동 경로를 출력한다.
