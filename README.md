# 3단계 - 자동차 경주
## 기능 목록 정리
- 사용자는 자동차 대수를 입력할 수 있다.
- 사용자는 시도햘 횟수를 입력할 수 있다.
- 자동차는 전진하거나 멈출 수 있다.
    - 전진 조건은 0에서 9사이 random 값을 구한 후 random 값이 4이상이어야 한다.
- 자동차 상태를 화면에 출력한다.

# 4단계 - 자동차 경주(우승자)
## 기능 목록 정리
- Car 객체에 이름 속성을 추가한다.
  - 이름 입력값은 6자 미만으로만 입력할 수 있게 처리한다.
  - 이름 입력 시, 콤마를 기준으로 구분한다. 
  - 실행 결과 출력 시, 자동차 이름을 추가한다.
- 자동차 게임 완료 후, 우승자를 출력한다. (공동 우승 가능)
