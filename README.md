# 자동차 경주 게임
## 3단계 - 자동차 경주
### 구현할 기능 목록
#### View
###### InputView
* 자동차 경주 게임 시작을 위한 데이터 입력
  * 자동차 n대
  * 시도할 횟수 m회
###### ResultView
* 자동차 경주 게임 결과 출력
  * 각 회차동안의 자동차의 상태 출력
    * 각 회차의 마지막 시점에 출력
#### Model
###### Car
* 현재 상태
  * 전진
  * 멈춤
* 이번 라운드 진행
  * 전진 or 멈춤
* 이번 라운드 결과 도출
  * 이번 라운드의 결과를 String 값으로 반환
#### Controller
###### RacingCarController
* 자동차 경주 게임 시작을 위한 데이터 입력값 받음
* 자동차 경주 게임 시작을 위한 데이터 세팅
* 자동차 경주 시작
* 자동차 경주 게임 결과 출력
#### Test
* 단위 테스트
  * 4이상일 경우에만 전진, 아닐 경우 멈춤
