# 자동차 경주 게임
## 진행 방법
* 자동차 경주 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 코드 리뷰 요청을 한다.
* 코드 리뷰 피드백에 대한 개선 작업을 하고 다시 PUSH한다.
* 모든 피드백을 완료하면 다음 단계를 도전하고 앞의 과정을 반복한다.

## 온라인 코드 리뷰 과정
* [텍스트와 이미지로 살펴보는 온라인 코드 리뷰 과정](https://github.com/next-step/nextstep-docs/tree/master/codereview)

## 요구사항 정리
### Step2
- 빈 문자열이 입력되면 0을 반환한다.
- null이 입력되면 0을 반환한다.
- 구분자가 없이 숫자 하나이면 그대로 반환한다.
- "," 로 구분자로 가지는 문자열을 split해서 수를 더한다.
- ":" 로 구분자로 가지는 문자열을 split해서 수를 더한다.
- 커스컴 구분자를 입력받아 구분해서 수를 더한다. (ex. //;\n1;2;3” 형식)
- 커스텀 구분자를 입력받은 형태가 아닌데 "," 혹은 ":" 구분자가 아닌 경우는  RuntimeException을 던진다.
- 음수를 전달하면 RuntimeException을 던진다.
- 커스컴 구분자를 입력받아 나눴는데 나눈 결과에 숫자 이외 문자가 들어 있으면  RuntimeException을 던진다.

### Step3
- 자동차 대수를 입력 받아야 한다.
- 자동차 대수를 입력 받을 때 숫자가 아니면  RuntimeException을 던진다.
- 자동차 대수를 입력 받아야 0 또는 음수일 경우 RuntimeException을 던진다.
- 주어진 횟수를 입력 받아야 한다.
- 주어진 횟수를 입력 받을 때 숫자가 아니면  RuntimeException을 던진다
- 주어진 횟수를 입력 받을 때 0 또는 음수일 경우 RuntimeException을 던진다.
- 주어진 횟수 동안 n대의 자동차는 전진 또는 멈춘다.
- 전진하는 조건은 Random한 확률로 움직이고 50%의 확률로 움직인다. (0에서 9 사이에서 random 값을 구한 후 random 값이 4이상일 경우)
- 자동차의 상태를 주어진 횟수만큼 출력한다.
